# Design Style Catalog

This file lists ALL available design styles with example screenshots. Even though this project uses a specific style (see CLAUDE.md → "Design Style"), this catalog exists so you can:
1. Reference other styles if the current one isn't working
2. Compare approaches when stuck
3. Suggest a style change to the developer if the chosen style doesn't fit the product

Use WebFetch to view any of these example URLs for visual inspiration.

---
